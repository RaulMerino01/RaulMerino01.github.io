Modelo de Referencia OSI

El modelo de referencia OSI se utiliza para entender y estructurar el flujo de comunicación entre dispositivos de red. Este modelo se compone de 7 capas: Física, Enlace de Datos, Red, Transporte, Sesion, Presentación y Aplicación. Cada una tiene una función específica dentro del proceso de comunicación.

Protocolos y Estándares

Los protocolos de comunicación permiten la transmisión de datos entre dispositivos de manera eficiente y segura. Algunos de los protocolos más importantes son:

  • HTTP/HTTPS: Utilizado para la transferencia de hipertexto en la web. Ver más sobre HTTP/HTTPS
  • FTP: Protocolo de transferencia de archivos entre sistemas remotos. Ver más sobre FTP
Leer más sobre los protocolos de comunicación

Dispositivos de Comunicación

Los dispositivos de comunicación permiten la interconexión de redes y la transmisión de datos. Algunos de los más comunes son:

  • Router: Conecta redes y dirige el tráfico de datos entre ellas. Ver más sobre Router
  • Switch: Interconecta dispositivos dentro de una red local (LAN). Ver más sobre Switch
  • Repetidor: Amplifica la señal para aumentar su alcance. Ver más sobre Repetidor
  • Punto de Acceso (AP): Permite la conexión inalámbrica de dispositivos a una red LAN. Ver más sobre Punto de Acceso
